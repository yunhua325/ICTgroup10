一、实验目的
本实验旨在通过编写和测试重复码编解码器及误码率计算程序，深入理解有噪信道编码的理论与实际应用。
二、主要程序及终端命令
1. 重复码编解码程序 repetitionCoder.py
- 编码命令：
  
  python repetitionCoder.py 
  encode LEN INPUT OUTPUT

  - LEN ：重复码长度（奇数，3~9）
  - INPUT ：待编码文件路径
  - OUTPUT ：编码后文件路径
- 解码命令：
  

  python repetitionCoder.py 
  decode INPUT OUTPUT

  - INPUT ：编码后文件路径
  - OUTPUT ：解码后文件路径
2. 误码率计算程序 calcErrorRate.py
- 误码率计算命令：

  python calcErrorRate.py 
  INPUT1 INPUT2 RESULT

  - INPUT1 、 INPUT2 ：待比较的两个文件路径
  - RESULT ：输出结果CSV文件路径
## 三、单元测试说明
### 1. 重复码编解码器测试 /unit-test/test_repetitionCoder.py
- 测试内容包括：
  - 编码与解码的正确性（不同重复码长度下，解码结果应与原始数据一致）
  - 多数表决机制的鲁棒性（即使部分比特被篡改，解码仍能恢复原始数据）
- 运行测试命令：

  python unit-test/
  test_repetitionCoder.py

### 2. 误码率计算程序测试 /unit-test/test_calcErrorRate.py
- 测试内容包括：
  - 两文件完全一致时误码率为0
  - 两文件完全不同、部分不同、长度不同等情况的误码率计算
  - 结果CSV文件输出格式校验
- 运行测试命令：

  python unit-test/
  test_calcErrorRate.py

 四、实验流程举例
1. 选择一个大于10KB的文件X，分别用3种不同的重复码长度编码：

   python repetitionCoder.py 
   encode 3 X X3.bin
   python repetitionCoder.py 
   encode 5 X X5.bin
   python repetitionCoder.py 
   encode 7 X X7.bin

2. 将编码后的文件通过不同错误概率的BSC信道（需结合实验3.1相关程序）：

   python byteChannel.py X3.
   bin Y_0.2_3.bin 0.2
   python byteChannel.py X5.
   bin Y_0.5_5.bin 0.5
   # 依次类推

3. 对信道输出文件进行解码：

   python repetitionCoder.py 
   decode Y_0.2_3.bin Z_0.2_3.
   bin
   python repetitionCoder.py 
   decode Y_0.5_5.bin Z_0.5_5.
   bin
   # 依次类推

4. 计算误码率：
   python calcErrorRate.py X 
   Z_0.2_3.bin result_0.2_3.
   csv
   python calcErrorRate.py X 
   Z_0.5_5.bin result_0.5_5.
   csv
   # 依次类推
